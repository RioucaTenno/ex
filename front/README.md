# ClubFlow - Система управления компьютерным клубом

Полнофункциональная система управления ресурсами компьютерного клуба с современным дизайном и интуитивным интерфейсом.

## 🎮 Особенности

- **Главная страница** - Интерактивная карта зала с цветовой индикацией статусов ПК
- **Бронирование** - Удобная система бронирования с выбором тарифа и дополнительных услуг
- **Личный кабинет** - Управление бронированиями, история посещений и бонусная программа
- **Панель администратора** - Мониторинг текущих сеансов и управление залом
- **Аналитика** - Подробные графики и отчёты для владельцев

## 🎨 Дизайн

- **Цветовая схема:**
  - Основной: Тёмно-синий (#1A237E)
  - Акцентный: Неоновый зелёный (#00FF88)
  - Нейтральный: Серый (#F5F5F5)
  
- **Типографика:** Inter (Light 14px, Regular 16px, Bold 24-32px)
- **Стиль:** Modern flat design с минимализмом и геймифицированными акцентами

## 📱 Страницы

### Для клиентов:
- **/** - Главная страница с картой зала и формой поиска
- **/tariffs** - Страница с подробным описанием тарифов
- **/booking/:pc_id** - Страница бронирования конкретного ПК
- **/profile** - Личный кабинет клиента

### Для администрации:
- **/admin/dashboard** - Панель администратора с текущими сеансами
- **/admin/analytics** - Аналитический дашборд с графиками

## 🚀 Быстрый старт

Приложение уже настроено и готово к работе. Используйте навигацию в хедере для перехода между страницами.

### Навигация:
1. Кликните на иконку **настроек** (шестерёнка) в хедере для доступа к админ-панели
2. Наведите курсор на компьютеры на карте зала для просмотра характеристик
3. Кликните "Забронировать" для перехода к странице бронирования

## 🎯 Основные компоненты

- **Интерактивная карта зала** - SVG-визуализация с цветовой индикацией
- **Система тарифов** - 4 тарифных плана с различными опциями
- **Бонусная программа** - Накопление и использование бонусных баллов
- **Графики и аналитика** - Recharts для визуализации данных
- **Адаптивный дизайн** - Mobile-first подход с 3 брейкпоинтами

## 📊 Технологии

- React 18.3.1
- React Router 7.13.0
- TypeScript
- Tailwind CSS 4
- Recharts для графиков
- Lucide React для иконок

## 🔧 Возможности администратора

- Мониторинг занятости в реальном времени
- Управление текущими сеансами
- Просмотр статистики и аналитики
- Экспорт отчётов в PDF и Excel
- Поиск клиентов по номеру телефона
